O CSS

1. O que é CSS?


CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata). O CSS é uma linguagem usada para dar estilo às páginas Web criadas com HTML. Enquanto o HTML serve para estruturar o conteúdo (textos, imagens, botões), o CSS é responsável pela aparência visual da página.


Exemplos do que o CSS faz:


• Mudar cores dos textos;
• Alterar o tamanho das letras;
• Definir cores de fundo;
• Posicionar elementos na página.


2. Importância do CSS no Desenvolvimento Web


• Melhora a aparência dos sites;
• Facilita a leitura do conteúdo;
• Separa conteúdo (HTML) do design (CSS);
• Facilita manutenção e actualizações;
• Melhora a experiência do utilizador.


3. Tipos de Folhas de Estilo CSS


3.1 CSS Inline


O CSS inline é aplicado directamente dentro da tag HTML.
Exemplo:
<p style="color: blue;">Este texto é azul</p>


Características:
• Simples e rápido;
• Para pequenas alterações;
• Não recomendado para páginas grandes.


3.2 CSS Interno (Internal CSS)


O CSS interno é escrito dentro da própria página HTML, usando a tag <style>.


Exemplo:
<style>
p { color: green; }
</style>


Características:
• Organiza melhor que o inline;
• Aplica estilo a vários elementos;
• Usado quando o estilo é apenas para uma página.
